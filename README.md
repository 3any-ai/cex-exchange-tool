# cex-exchange-tool
- 本版本只支持OKX
- 本版本还有很多不完善的地方，并未经过大量测试，使用者自行承担风险
- 交易账户不要存放过多资金，以免出现bug,或配置不当，消耗完交易账户中所有资金
- 尽量使用子账户，少量资金试用
